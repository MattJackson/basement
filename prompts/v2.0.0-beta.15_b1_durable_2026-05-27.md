# v2.0.0-beta.15 — B1 durable: backend auto-detects Garage version on seed

## What you're producing

Modify `cmd/basement-server/main.go:111-152` so that the auto-seed
of the "default" connection probes the admin URL before deciding
between `garage` (v2) and `garage-v1`. Without this, operators who
run Garage v1 but set `BASEMENT_DRIVER=garage` (the default) end up
with a permanently Unavailable cluster — every deploy re-creates it
with the wrong driver.

## Background

The operator's deploy hits this every backend restart:
1. Container starts; persistent connection store has 0 entries
   (apparently the store doesn't survive container restarts).
2. main.go:111 sees count==0 and `cfg.Driver.Name="garage"`,
   auto-seeds a default connection with driver=garage.
3. The actual Garage server at `cfg.Driver.Garage.AdminURL` only
   supports the v1 admin API; v2 calls fail with
   `{"code":"InvalidRequest","message":"Bad request: Unknown API endpoint: GET /v2/ListBuckets"}`.
4. /admin/buckets, /admin/usage, /admin/clusters all error out.
5. Senior PATCHes driver=garage-v1 by hand; works until next deploy.

Endpoint probes (from manual testing against 10.1.7.10:3903):
- `GET /v1/health` on a v1 server → `403 AccessDenied` (endpoint
  EXISTS, just needs auth)
- `GET /v2/GetClusterStatus` on a v1 server → `400 InvalidRequest:
  Unknown API endpoint`

So: if `/v2/GetClusterStatus` returns a body containing `"Unknown API
endpoint"`, the server is v1. Otherwise (e.g. 200 or 401/403 with a
different message body), assume v2.

## Implementation

### 1. Add a probe helper

In `cmd/basement-server/main.go`, near the top of the file (above
`func main()`), add a tiny helper:

```go
// detectGarageVersion probes the admin URL to choose between
// driver=garage (v2) and driver=garage-v1. Returns the input default
// if the probe is inconclusive (network error, etc) — the seed will
// then fall through with whatever the operator configured.
//
// Probe semantics, based on actual Garage server behavior:
//   - GET /v2/GetClusterStatus on a v1-only server returns
//     {"code":"InvalidRequest","message":"Bad request: Unknown API endpoint..."}
//     We detect this body marker and report v1.
//   - On a true v2 server, /v2/GetClusterStatus returns either 200
//     (no auth) or 401/403 with a different error body. We report v2.
func detectGarageVersion(adminURL, fallback string) string {
	if adminURL == "" {
		return fallback
	}
	client := &http.Client{Timeout: 3 * time.Second}
	resp, err := client.Get(strings.TrimRight(adminURL, "/") + "/v2/GetClusterStatus")
	if err != nil {
		// Network error — fall back without changing the operator's choice.
		slog.Warn("detectGarageVersion: probe failed; falling back", "err", err, "fallback", fallback)
		return fallback
	}
	defer resp.Body.Close()
	body, _ := io.ReadAll(io.LimitReader(resp.Body, 4096))
	bodyStr := string(body)
	if strings.Contains(bodyStr, "Unknown API endpoint") {
		slog.Info("detectGarageVersion: detected Garage v1 (admin URL rejects /v2/* endpoints)", "url", adminURL)
		return "garage-v1"
	}
	slog.Info("detectGarageVersion: detected Garage v2 (admin URL accepts /v2/* shape)", "url", adminURL)
	return "garage"
}
```

You'll need to add imports if not already present:
- `"io"`
- `"net/http"`
- `"strings"`
- `"time"`

### 2. Wire it into the seed

At main.go line ~138 (just before the `newConn := store.Connection{...}`
construction), if `cfg.Driver.Name` is `garage` or `garage-v1`, run
the probe and use the result. Pseudocode:

```go
driverName := cfg.Driver.Name
if driverName == "garage" || driverName == "garage-v1" {
    driverName = detectGarageVersion(cfg.Driver.Garage.AdminURL, driverName)
}

newConn := store.Connection{
    Label:  "default",
    Driver: driverName,   // <-- was cfg.Driver.Name; now the probed value
    Config: connConfig,
    Owner:  "org",
}
```

Don't touch the `connConfig` block — it already handles both garage
and garage-v1 the same way (both expect admin_url + admin_token).

### 3. Add a unit test

Create or extend `cmd/basement-server/main_test.go` with a test that
hits a small `httptest.Server`:

```go
func TestDetectGarageVersion(t *testing.T) {
    // v1 server: returns "Unknown API endpoint" on /v2/*
    v1srv := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        if strings.HasPrefix(r.URL.Path, "/v2/") {
            w.WriteHeader(400)
            fmt.Fprintln(w, `{"code":"InvalidRequest","message":"Bad request: Unknown API endpoint: GET /v2/GetClusterStatus"}`)
            return
        }
        w.WriteHeader(200)
    }))
    defer v1srv.Close()
    if got := detectGarageVersion(v1srv.URL, "garage"); got != "garage-v1" {
        t.Errorf("v1 server: want garage-v1, got %s", got)
    }

    // v2 server: returns 401 or 200 on /v2/*
    v2srv := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        if strings.HasPrefix(r.URL.Path, "/v2/") {
            w.WriteHeader(401)
            fmt.Fprintln(w, `{"code":"AccessDenied","message":"Forbidden: Authorization token must be provided"}`)
            return
        }
        w.WriteHeader(200)
    }))
    defer v2srv.Close()
    if got := detectGarageVersion(v2srv.URL, "garage"); got != "garage" {
        t.Errorf("v2 server: want garage, got %s", got)
    }

    // Network failure: falls back to caller's choice
    if got := detectGarageVersion("http://127.0.0.1:1", "garage-v1"); got != "garage-v1" {
        t.Errorf("net failure: want fallback garage-v1, got %s", got)
    }
}
```
